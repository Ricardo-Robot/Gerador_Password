##  Gerador de Password 
Um pequeno app feito com JavaScript puro, para gerar passwords automaticamente usando <b>LETRAS_MAIUSCULAS</b>, <b>NUMEROS</b> e <b>SIMBULOS</b>.
Perfeito para quem está a aprender a codificar. <br>
